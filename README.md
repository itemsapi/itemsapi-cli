# ItemsAPI application generator

## Installation

```sh
$ npm install -g itemsapi-cli
```

## Commands

Generating new application 

```bash
$ itemsapi generate my-app 
```

Importing data
```bash
$ itemsapi import -c zouespow -f data.json --api=http://localhost:5000/api/v1 -l 1000
```
